Corso has been deployed!

1. Verify the installation:
   kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/name={{ include "corso.name" . }}

2. Check the controller (if enabled):
   kubectl logs -n {{ .Release.Namespace }} -l app.kubernetes.io/component=controller

3. Create an allowlist:
   cat <<EOF | kubectl apply -f -
   apiVersion: corso.io/v1alpha1
   kind: BPFProgramAllowlist
   metadata:
     name: default
     namespace: {{ .Release.Namespace }}
   spec:
     defaultAction: alert
     ignoreKnownDaemons: true
     programs:
       - name: my_tracer
         type: kprobe
   EOF

4. View metrics:
   kubectl port-forward -n {{ .Release.Namespace }} svc/{{ include "corso.fullname" . }}-metrics 9090:9090
   curl http://localhost:9090/metrics
